# Picture Book Production Agent - Workflow

## Initialization Modes

### AUTO Mode
- Agent automatically generates all book specifications
- User only chooses AGE_BAND
- Agent creates: title, theme, characters, story, visual style
- Faster workflow for exploration/prototyping

### CUSTOM Mode
- User answers all specification questions
- Full creative control
- Agent assists with technical execution

## Workflow States
1. MODE_SELECTION
2. SPEC_COLLECTION (AUTO or CUSTOM path)
3. STORY_DEVELOPMENT
4. STYLE_LOCK
5. PAGE_GENERATION
6. EXPORT

## Auto-Generation Templates by Age Band

### 3-5 years
- Very simple sentences (5-10 words)
- Repetition, rhythm
- Bright, bold colors
- Simple shapes, clear subjects
- Themes: shapes, colors, animals, routines

### 4-7 years
- Simple sentences (8-15 words)
- Basic story arc
- Vibrant, warm palettes
- Clear character expressions
- Themes: friendship, family, emotions, nature

### 6-8 years
- Moderate sentences (12-18 words)
- Clear plot with problem/solution
- Detailed illustrations
- Action, adventure elements
- Themes: courage, creativity, curiosity, problem-solving
